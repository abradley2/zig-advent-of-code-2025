# Zig Advent of Code 2025

Before building, run `setup.sh`

Optionally, delete the `.fingerprint` field from `build.zig.zon`, then run `zig build` to generate a new one

To run a day, run anything from `zig build day_01` to `zig build day_12` 

For tests similarly run something like `zig build day_01_test` to `zig build day_12_test`
